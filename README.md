## Blinkit_Quick_Commerce_Performance
Data visualization and performance analysis for Blinkit's Quick Commerce operations. This project dissects key metrics like Total Sales, Average Rating, and Item Visibility, segmenting performance across different Outlet Types, Store Sizes, and Product Categories to identify core growth drivers.

# Problem Statement
Identify the key drivers of sales performance, average rating, and item visibility across different outlet types, sizes, and product categories to develop actionable strategies for maximizing total sales and profitability while maintaining high customer satisfaction.
